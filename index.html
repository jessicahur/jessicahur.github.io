Please click the button on the bottom to start the game or keep reading if you are curious about what javascrip does on this page.

Asking for player's name
  1. JS receives user input through the prompt() method which asks "What is your name?"
  2. JS stores that input in a variabe named 'name'
  3. Then JS will log this value onto the console for debugging purposes.
  4. JS will prompt an 'alert' message box with a greeting message with the value of 'name' concatenated with "Nice meeting you " and "! Let's play a game. Please answer 3 questions about me using yes and no only."
Asking questions and evaluating answers
  1. Once you enter your answer, JS will check if you enter anything. Emtpy answer is not accepted.
  2. JS will seek to confirm if you are happy with your answer. If you want to change your answer, click Cancel on the confirm box and it will bring you back to the answer prompt. If you are happy, just click OK
  3. JS will convert your answer to upper case. Then it will compare your answer against the answer keys stored in an array. If the two are strictly match, then JS will update the number of correct answers. If not, then JS will 'alert' you with a sorry message.
